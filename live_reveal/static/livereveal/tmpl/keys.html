• alt + r: Enter/Exit RISE
  • w: Toogle overview mode
  • home: First slide
  • end: Last slide
  • space bar: Next
  • shift + space bar: Previous
  • pgup: Up
  • pgdn: Down
  • left arrow: Left
  • right arrow: Right
  • black screen: Period (or forward slash)
NOTE: You have to use this shortcuts in command mode.
